# 사용자 규칙

## 언어 설정
- **모든 응답은 한글로 작성해주세요.**
- 시스템 메시지, 설명, 안내, 질문 등 모든 텍스트는 한국어로 표시합니다.

- 기술 용어는 필요시 영어 원문을 병기할 수 있습니다 (예: "컴포넌트(component)").
- 마크다운 형식을 사용하여 가독성을 높입니다.
